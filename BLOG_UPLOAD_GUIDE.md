## STEP 4 ‚Äî Markdown Rules (Body Field)

The body is written in Markdown. Here are all the rules you need to know:

---

### Headings
```markdown
## This is a Section Heading       ‚Üê Use this (H2)
### This is a Sub-heading          ‚Üê Use for sub-points
```
Never use `# Heading` (H1) ‚Äî the title field handles H1 automatically.

---

### Bold Text
```markdown
Our model shows **67%** win probability vs. market implied **52%**.
```
Wrap any key stat, odds figure, or important term in `**double asterisks**`.

---

### Blockquotes (for power stats/key insights)
```markdown
> **Key Stat:** Teams trailing at half-time in the last 10 H2H meetings have covered the handicap **8 times out of 10**.
```
Use once per major section for the most impactful stat.

---

For **every image** in the article, paste this exact line at the point where it should appear:

```markdown

```

### The FACT_CTA Block (Random Fact Card)
Place this tag **once per article**, ideally after Section 5 (Risks) and before the Bottom Line:

```markdown

```

This will automatically display a beautiful animated fact card on the live site. No extra work needed ‚Äî it's already built in the website.

### Image Placement by Article Length

| Images in article | Where to place them |
|---|---|
| 2 images | After intro paragraph + Before conclusion |
| 3 images | After intro + Mid-article (data section) + Before conclusion |
| 4 images | After intro + After data section + After edge section + Before conclusion |

Always spread images evenly. Never stack two images back-to-back with no text between them.

## COMMON MISTAKES ‚Äî Don't do these

| ‚ùå Wrong | ‚úÖ Right |
|---|---|
| Using spaces in image filenames | Use hyphens: `my-image.jpg` |
| Using `# Heading` (H1) in the body | Use `## Heading` (H2) |
| Uploading images AFTER writing | Upload images FIRST, note filenames |
| Writing more than 4 tags | Exactly 4 tags, no more |
| Excerpt longer than 160 characters | Keep it tight ‚Äî 1‚Äì2 sentences max |
| Missing `[[FACT_CTA]]` | Place it once between sections |
| Stacking two images with no text | Always have a paragraph between images |
